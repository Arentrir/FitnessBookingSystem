## Описание на проекта

Fitness Booking System е уеб приложение, разработено с ASP.NET Core MVC, чиято цел е да улесни записването за фитнес класове, управлението на графици и визуализирането на треньори.  
Проектът включва поддръжка на роли (Admin / Trainer / User), система за резервации, управление на класове и треньори, качване на снимки и модерна визуална част.
